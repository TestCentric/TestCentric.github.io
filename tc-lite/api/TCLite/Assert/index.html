Assert Class

Summary

The Assert class contains a collection of static methods that implement the assertions used in TC-Lite.
Namespace
TCLite
Base Types
  • object
graph BT Type-->Base0["object"] Type["Assert"] class Type type-node

Syntax

public abstract class Assert

Remarks

We don't actually want any instances of this object, but some people like to inherit from it to add other static methods. Hence, the use of abstract in the declaration.
